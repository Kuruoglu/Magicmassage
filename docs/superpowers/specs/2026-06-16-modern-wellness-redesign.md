# Modern Wellness Redesign

## Goal

Replace the static recolored layout with a more modern boutique wellness
composition while preserving the current public routes, localized content, and
real project photography.

## Direction

Use a cinematic, editorial layout rather than a flat template:

- dark emerald and warm brown depth in the hero;
- ivory content surfaces with gold details;
- larger image-led sections;
- varied card sizes and asymmetry;
- subtle CSS-only motion and hover states;
- refined decorative lines inspired by the salon interior;
- no generated salon imagery in production.

## Scope

This slice changes the public visual presentation only. It covers:

- home hero composition;
- trust strip;
- featured services section;
- about preview;
- booking call to action;
- shared header/footer polish;
- visual styling for Services, About, and Contacts pages.

It does not add new business data, booking logic, admin features, blog routes,
payments, or third-party animation libraries.

## Acceptance Criteria

- The site should no longer feel like a simple stacked template.
- The home page should have stronger visual rhythm and more varied section
  composition.
- The palette should stay close to the supplied reference: emerald, ivory,
  warm brown, and gold.
- Existing routes and locale switching must keep working.
- The implementation must pass tests, lint, type checking, production build,
  and a real browser desktop/mobile check.
